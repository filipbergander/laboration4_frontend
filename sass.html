En hemsida skapad genom CSS-preprocessor SASS.

För- och nackdelar med SASS:

  • Jag tyckte det var enkelt att komma igång. Jag gjorde på samma vis som tidigare uppgifter. Skapade en grund genom npm create vite@latest och fördelning av filer inom huvudmappen. Exempelvis node_modules till .gitignore samt vite.config.js med övriga sidor också vidare.
  • Jag skapade Sass-fil genom npm install sass --save-dev.
