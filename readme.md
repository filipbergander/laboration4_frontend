# Laboration 4 - Animeringar

En laboration inom Frontend-utveckling där syftet är att skapa animeringar. Jag valde att skapa en knapp som är rörlig på skärmen för att starta en demonstration. 
Vid klick på att starta demonstrationen finns det animeringar genom en roterande laddningsikon, en geting som flyger ovanpå skärmen, skiftande färger på tummar för att gilla/inte gilla innehållet. Vid klick på tummarna åker de upp eller ned genom translate i Y-led. Olika färger på element också vidare, samt hover-effekter.

Webbplats: https://lab4frontend-fb.netlify.app/
